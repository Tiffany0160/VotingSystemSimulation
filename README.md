# Voting System

A simple Java console-based voting system that allows users to vote for candidates, view results, and display the winner.

## Features
- Vote for a candidate
- View vote results
- Display the winning candidate
- Simple menu-driven system

## Candidates
- Alice
- Bob
- Charlie

## Technologies Used
- Java
- Scanner class
- Arrays

## How to Run
1. Open the project in a Java IDE (NetBeans, IntelliJ, Eclipse, etc.)
2. Run the `VotingSystem.java` file
3. Use the menu to vote, view results, or exit
